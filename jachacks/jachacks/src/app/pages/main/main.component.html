Beginner Enterprise

Security Toolkit

This multi use tool leverages AI to accomplish multiple cybersecurity-related tasks to facilitate the job of cybersecurity analyst, cybersecurity students and developers alike. You can find a detailed description of every tool down below.

Password Checker

The password checker checks the password against known password lists to determine if the password is secure. Then, it uses AI to give a security score to the password and suggest changes to the user to make it more secure.

Phishing Detector & Creator

The phishing detector lets you upload a .eml file or enter manually the email information (sender email, receiver email and email content). It will then analyze the email using AI to scan for different indicators of untrustworthiness.

The phishing creator uses AI to helps create email templates using different attack strategies to facilitate enterprise security training. Analysts can use this tool to easily create phishing emails they can then send to employees to test their security awareness.

Code Analyzer

The code analyzer analyzes a snippet of code for potential vulnerabilities. It uses AI to point out the vulnerabilities and offer solutions and examples of how to fix them and make the code more secure and less prone to attacks.

Report Generator

Considering that a large part of cybersecurity is paperwork, the report generator allows for the creation of several types of cybersecurity reports, based on a summary of noted events. It also allows to spin the events in either a positive or a negative light, giving flexibility to adjust one's work depending on the audience.
